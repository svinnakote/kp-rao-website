# KP Rao Advisory Website Prototype

## Project Goal
Build a premium, simple, trust-oriented website for an independent financial advisor.

This is NOT a generic fintech product.
This is a personal advisory practice website focused on credibility, client acquisition, and thought leadership.

## Brand Positioning
KP Rao Financial Advisory

Tagline:
Plan • Grow • Protect • Prosper

Brand attributes:
- Trustworthy
- Calm
- Premium
- Human
- Professional
- Simple, not flashy

## Target Audience
- Families
- Retirees
- Working professionals
- HNI clients
- Existing referral network prospects

## Primary Goal of Site
1. Build trust
2. Generate consultation leads
3. Showcase expertise
4. Support future content publishing

## MVP Site Sections
- Hero
- Services
- Why KP Rao
- Insights
- Contact

## Hero Copy
Headline:
Trusted Financial Guidance
For Families & Professionals

Subtext:
Helping clients make informed financial decisions.

CTA:
Book Consultation

## Core Services
- Tax Planning
- Investment Advisory
- Retirement Planning

## Visual Direction
Style:
- Minimal
- Premium advisory firm aesthetic
- Similar feel to boutique wealth management firms

Colors:
- Deep navy
- Teal
- Gold accents only

## Tech Stack
Initial prototype:
- HTML
- CSS
- Optional light JavaScript only

Keep it simple.
No frameworks unless explicitly requested.

## Constraints
- Fast loading
- Responsive
- Clean semantic HTML
- Accessible
- Easy to later migrate into WordPress

## Future Direction
Content may later be published via:
- Medium or Substack
- LinkedIn distribution
- Website may surface latest insights
